# Zsh Error Sound

Plays the Fahhhhh sound whenever an error occurs. This is currently ONLY for zsh

## Install automatically

```bash
curl -sSL https://raw.githubusercontent.com/yourusername/zsh-error-sound/main/install.sh | bash
```

## Install manually

```bash
git clone https://github.com/yourusername/zsh-error-sound.git
cd zsh-error-sound
chmod +x install.sh
./install.sh
```

## Uninstall
```bash
./uninstall.sh
```
